# Техническое задание
## Необходимо создать приложение «Ветеринарная Клиника» на Java

Приложение может быть сделано как в консольном виде, так и в браузере или в
виде приложения на Android, iOS. Строгих требований к реализации и
функциональности нет, подойди к заданию творчески! Главное для нас —
посмотреть, как ты выполнишь задачу и какие инструменты в разработке
используешь.

*Функции:*
 - Новый пациент.
 - Запись к врачу.
 - Медицинские карточки.

*Обязательно:*
 - Создать схему базы данных для описанных функций.
 - Реализовать приложение для одной из функций, можно упрощенно.

*Желательно:*
 - Создать объектную модель приложения.

Идеи для реализации:
Функции: добавить, удалить, редактировать.
Постарайся показать свои сильные стороны: мы ценим не только результат, но и
твой подход к решению задачи. Удачи, у тебя всё получится!
